A Propos

Qui sommes-nous ?

Ce site web a été réalisé par Youri Gicquel, Manon Delanoy, Thomas Martin et Romain Favre, étudiants d'Epiteh Lyon.

Epitech est une école d'expertise informatique et offre à ses étudiants un cursus en 5 ans. Tous les membres de ce projet sont en fin de troisième année.

Pourquoi ce projet ?

Cette interface web a été developpée dans le cadre d'un projet de fin de troisième année, impliquant un travail de groupe de plusieurs mois. Nous étions désireux de découvrir les technologies web et de developper un outil utile et performant.

C'est donc ainsi qu'est né notre projet.

Le projet en quelques mots ...

Son fonctionnement est simple: Il suffit d'entrer le code voulu dans l'éditeur prévu à cet effet (voir Editeur) et d'envoyer son code au compilateur via le bouton COMPILER. Si le code est correct, la mention OK apparaitra, sinon KO.
